# CYBERHARI Labs 🛡️

**CYBERHARI Labs** is an elite, interactive cybersecurity learning platform designed to bridge the gap between theoretical knowledge and practical execution. It provides a safe, sandbox environment for running real-world Python security tools through a sleek, high-tech web interface.

![CYBERHARI Banner](https://raw.githubusercontent.com/placeholder/banner.png)

## 🚀 Features

- **24 Specialized Security Tools**: From network reconnaissance to vulnerability assessment.
- **Interactive Terminal**: Real-time execution of Python scripts with live output.
- **Learning Modules**: Structured modules covering Network Recon, Web Security, Cryptography, and more.
- **Lab History**: Track and review your previous lab activities.
- **Modern UI/UX**: Premium design with matrix backgrounds, HUD overlays, and responsive themes.

## 💻 Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Custom Design System), JavaScript (ES6+)
- **Backend**: FastAPI (Python 3.9+)
- **Database**: SQLAlchemy with SQLite
- **Tools**: Pure Python 3

## 🏁 Getting Started

### Prerequisites

- Python 3.9 or higher
- `pip` (Python package manager)

### Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/cyberhari-labs.git
   cd cyberhari-labs
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Start the Backend Server**:
   ```bash
   python server/main.py
   ```

4. **Launch the Interface**:
   Simply open `client/index.html` in any modern web browser.
